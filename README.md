# Guess the Number

A command-line guessing game where the computer picks a number and the player tries to guess it.

## What It Does

The computer picks a random number between 1 and 100. The player types in guesses, and after each guess the computer gives a hint: "too high" or "too low". The game keeps going until the player guesses correctly, then tells them how many attempts it took.
