# Task List for Guess the Number

1. **Set up the project skeleton** — Create the main JavaScript file and verify Node.js can run it. Learn: project structure, running a script with Node.js.

2. **Generate a random number** — Write code to pick a random integer between 1 and 100 and print it to the terminal (temporarily, so you can verify it works). Learn: variables, `Math.random()`, `Math.floor()`.

3. **Read user input from the terminal** — Use Node's built-in `readline` module to prompt the player for a guess and print it back. Learn: how to accept keyboard input in Node, callbacks or events.

4. **Build the hint logic** — Compare the player's guess to the secret number. Print "Too high!" or "Too low!" depending on the result. Learn: if/else conditionals, comparison operators (`>`, `<`, `===`).

5. **Create the game loop** — Keep asking for guesses repeatedly until the player gets it right. Print "You got it!" when they guess correctly. Learn: looping patterns, controlling program flow.

6. **Track attempts and display score** — Count each guess and display the number of attempts when the player wins. Learn: incrementing a counter variable, string interpolation (template literals).

7. **Add a welcome message and instructions** — Print a friendly message at the start telling the player what the game is and what to do. Learn: user experience, writing clear prompts.

8. **Add input validation** — Handle cases where the player types something that isn't a valid number. Show a friendly error message and let them try again without losing a guess. Learn: type checking, `isNaN()`, graceful error handling.
